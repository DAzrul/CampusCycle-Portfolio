# 🛒 CampusCycle (Gingerbread - Nougat)
*A Peer-to-Peer Marketplace Application for University Students*

CampusCycle adalah aplikasi mudah alih Android (Native) yang direka khas untuk memudahkan transaksi jual beli secara bersemuka (face-to-face) dalam kalangan pelajar universiti. 

## ✨ Ciri-Ciri Utama (Features)
- 🏬 **Student Marketplace:** Platform jual beli barang (pre-loved, buku teks, makanan) khas untuk pelajar.
- 👨‍💻 **Admin Monitoring System:** Sistem pemantauan untuk menguruskan fakulti dan aktiviti pengguna dalam aplikasi.
- 🔐 **Secure Authentication:** Log masuk dan integrasi pangkalan data masa nyata (real-time).

## 🛠️ Tech Stack
- **Language/Framework:** Kotlin (Native Android)
- **Backend/Database:** Firebase
- **Architecture & Design:** Entity-Relationship Modeling (EERD)
- **Tools:** Android Studio, Git
