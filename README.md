# 🛒 CampusCycle (Gingerbread - Nougat)
*A Peer-to-Peer Marketplace Application for University Students*

CampusCycle is a native Android mobile application designed to facilitate face-to-face marketplace transactions among university students.

## ✨ Key Features
- 🏬 **Student Marketplace:** A dedicated platform for students to buy and sell items (pre-loved goods, textbooks, food).
- 👨‍💻 **Admin Monitoring System:** An administrative dashboard to manage faculties and monitor user activities within the app.
- 🔐 **Secure Authentication:** Secure login and real-time database integration.

## 🛠️ Tech Stack
- **Language/Framework:** Kotlin (Native Android)
- **Backend/Database:** Firebase
- **Architecture:** Entity-Relationship Modeling (EERD)
- **Tools:** Android Studio, Git
